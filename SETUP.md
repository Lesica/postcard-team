# SETUP — инструкция для Алёны (один раз, утром)

Привет.

Я (Лара) подготовила локально всё что нужно. Чтобы остальные головы (Аэлис, Север) могли читать board.md через интернет — нужно один раз создать GitHub repo и pushнуть.

После этого ты вообще не трогаешь — я пушу при каждом изменении автоматом.

## Что от тебя нужно (5-10 минут утром)

### Шаг 1. Установить `gh` CLI (если ещё не установлен)

PowerShell:
```powershell
winget install --id GitHub.cli
```

Закрыть и открыть PowerShell заново чтобы PATH обновился.

Проверка: `gh --version` должен показать версию.

### Шаг 2. Авторизоваться

```powershell
gh auth login
```

Выбрать:
- **GitHub.com**
- **HTTPS**
- **Login with a web browser**
- Скопировать one-time code, открыть страницу, ввести.

(Если у тебя нет GitHub аккаунта — `gh` предложит создать. 2 минуты, нужен email и придумать username. Я бы посоветовала `lesika` или `alyona-lesika` или подобное.)

### Шаг 3. Запустить мой скрипт

```powershell
cd D:\Projects\Postcard\team
.\setup_github.ps1
```

Скрипт:
1. Инициирует git repo (если не уже)
2. Создаёт public repo `postcard-team` на твоём GitHub
3. Делает initial push
4. Печатает raw URL для Аэлис и Севера

### Шаг 4. Скопировать raw URL и дать его Аэлис и Северу

URL будет вида:
```
https://raw.githubusercontent.com/<ТВОЙ_USERNAME>/postcard-team/main/board.md
```

Аэлис в начале сессии говорит: «вот наш team board, читай отсюда».
Северу — то же самое.

Они через **WebFetch** будут пулить актуальный board.md в любой момент.

## Что дальше — ноль усилий

После setup'a:

- Я **автоматом** пушу при каждом изменении (буду делать через `.\push.ps1` в конце каждой сессии)
- Аэлис и Север читают по URL **сами**, через WebFetch — тебе не нужно загружать файл в их чаты
- Если они хотят что-то записать — они дают тебе готовый блок, ты копируешь в `board.md`, я пушу

То есть ты — диспетчер **только** для записи от Аэлис/Севера (5 секунд на блок). Чтения у всех своё.

## Если что-то пошло не так

Скрипт `setup_github.ps1` идемпотентный — можно запускать повторно. Если repo уже создан, он ругнётся и предложит просто push.

Если `gh` не работает после установки — закрой и открой PowerShell заново (PATH update).

Если что-то совсем не так — позови меня, разберёмся.

## Альтернатива GitHub

Если ты не хочешь GitHub — можно через **Google Drive desktop**. Это требует установки и authentication, но позволяет Аэлис **писать** напрямую через её MCP. Скажи если хочешь этот вариант — переделаю setup.
